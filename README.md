# Integrated-Patient-Care-Management-System
Integrated Patient Care Management System - Our Training Project for Infosys Springboard

## Milestone 1 Dashboard
A visually rich dashboard built with Streamlit, styled with custom CSS to match our exact design specs.

### How to Run
1. Ensure your virtual environment is active.
2. Install dependencies (if not already done): `pip install streamlit python-dotenv langchain-groq langchain-core`
3. Run the application:
   ```bash
   streamlit run main.py
   ```
4. Optional: Configure your `.env` file with a `GROQ_API_KEY` (see `CONTEXT/API_KEY_GUIDE.md`) to enable live LLM integration.
